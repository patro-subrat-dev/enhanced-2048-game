🎓 How to Play Enhanced 2048

1. Basic Movement

Use arrow keys (↑↓←→) or swipe gestures to move tiles in any direction.

2. Merging Tiles

When two tiles with the same number touch, they merge into one tile with double the value.

3. Building Combos

Chain multiple merges quickly (within 2 seconds) to build combos and earn bonus points!

4. Power-Ups

Use special abilities like bombs, extra undos, and score multipliers strategically.

5. Game Modes

Try different modes: Classic, Time Attack, Zen, or Challenge for varied gameplay.
